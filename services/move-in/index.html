What's included

Every clean follows the same checklist.

Kitchen

  • Inside + outside ALL cabinets + drawers
  • Inside oven (degrease + scrub)
  • Inside fridge + freezer (defrosted ok)
  • Inside dishwasher
  • Inside microwave
  • Polish all appliances
  • Sanitize counters + backsplash
  • Detailed sink + faucet
  • Sweep + mop floor
  • Hand-wipe baseboards

Bathrooms

  • Inside + outside vanity drawers
  • Inside medicine cabinet
  • Disinfect tub, shower, toilet
  • Detailed tile grout
  • Polish chrome + glass
  • Detailed sink + faucet
  • Sweep + mop
  • Hand-wipe baseboards

Bedrooms

  • Inside + outside closet shelves
  • Wipe inside drawers (built-ins)
  • Wipe windowsills + tracks
  • Vacuum / mop floor
  • Hand-wipe baseboards
  • Dust ceiling fans + fixtures

Whole home

  • Sanitize every light switch + door handle
  • Wipe interior doors
  • Detailed cobweb removal
  • Inside-window glass + sills
  • Vacuum / mop ALL floors
  • Wipe baseboards every room
  • Empty + wipe any built-in storage

What's not included

  • Carpet shampoo / steam (recommend separate carpet service)
  • Window exteriors / 2nd-story windows
  • Wall washing (add-on per room)
  • Garage, attic, basement crawl space
  • Painting, repairs, hauling

Need any of these? They're available as add-ons in the quote tool — you'll see the price live before you book.
