What's included

Every clean follows the same checklist.

Kitchen

  • Dust + wipe all surfaces
  • Sanitize counters + sink
  • Wipe stove top + microwave exterior
  • Wipe outside of major appliances
  • Empty trash + reline
  • Sweep + mop floor
  • Spot-clean cabinet fronts

Bathrooms

  • Disinfect counters, sink, faucets
  • Scrub toilet inside + out
  • Clean tub + shower
  • Streak-free mirrors + glass
  • Sweep + mop floor
  • Empty trash + replace liner
  • Restock if guest soap/towels in place

Bedrooms

  • Dust all flat surfaces
  • Make beds (linens stay)
  • Vacuum carpet / dust hardwood
  • Wipe baseboards (visible)
  • Empty trash

Living areas

  • Dust all surfaces + electronics
  • Vacuum carpet + furniture
  • Mop hardwood / tile
  • Sanitize light switches + door handles
  • Wipe inside windows (sills + ledges)

What's not included

  • Inside oven, fridge, or cabinets (add as add-on)
  • Wall washing or scuff removal (add-on)
  • Window exteriors or 2nd-story windows
  • Heavy-duty tile/grout deep-scrub (deep clean only)
  • Laundry, dishes, or pet waste

Need any of these? They're available as add-ons in the quote tool — you'll see the price live before you book.
