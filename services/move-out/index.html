What's included

Every clean follows the same checklist.

Kitchen

  • Inside oven (degrease + scrub)
  • Inside fridge + freezer
  • Inside dishwasher
  • Inside + outside ALL cabinets + drawers
  • Inside microwave
  • Polish all appliances
  • Sanitize counters + backsplash
  • Sink + faucet detail
  • Sweep + mop
  • Hand-wipe baseboards

Bathrooms

  • Inside + outside vanity, drawers, medicine cabinet
  • Tub, shower, toilet — detailed disinfect
  • Tile grout scrub
  • Polish chrome + glass
  • Sink + faucet detail
  • Sweep + mop
  • Hand-wipe baseboards

Bedrooms

  • Inside + outside closets (every shelf, every rod)
  • Inside any built-in drawers
  • Vacuum / mop floor
  • Hand-wipe baseboards
  • Wipe windowsills + tracks
  • Dust ceiling fans + light fixtures

Whole home

  • Sanitize every switch + door handle
  • Wipe interior doors + frames
  • Detailed cobweb removal
  • Inside-window glass + sills
  • Vacuum / mop ALL floors
  • Wipe baseboards every room
  • Empty + wipe inside any built-in storage
  • Inspect for missed items + final walk-through

What's not included

  • Carpet shampoo / steam (separate carpet service if needed)
  • Wall washing (add-on per room)
  • Window exteriors / 2nd-story
  • Hauling left-behind items
  • Repair work or painting

Need any of these? They're available as add-ons in the quote tool — you'll see the price live before you book.
