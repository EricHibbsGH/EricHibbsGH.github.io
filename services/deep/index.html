What's included

Every clean follows the same checklist.

Kitchen

  • Everything in standard, plus:
  • Detailed appliance exteriors (front, sides, top)
  • Inside microwave + handle
  • Polish stainless steel grain-direction
  • Edge-clean tile grout
  • Hand-wipe cabinet fronts (top + bottom)
  • Wipe range hood + filter
  • Behind small countertop appliances

Bathrooms

  • Everything in standard, plus:
  • Detailed tile grout scrub
  • Hard-water removal (faucets, glass)
  • Wipe baseboards + trim
  • Hand-wash inside cabinet doors
  • Wipe vent fan covers
  • Polish chrome / brushed nickel
  • Spot-treat tub caulking

Bedrooms

  • Everything in standard, plus:
  • Wipe baseboards + door frames
  • Dust ceiling fans + light fixtures
  • Wipe inside windowsills + tracks
  • Vacuum under bed (where reachable)
  • Spot-clean upholstery (no shampoo)

Living areas

  • Everything in standard, plus:
  • Detailed baseboards + trim
  • Hand-wipe ceiling fans + light fixtures
  • Vacuum upholstery
  • Wipe inside windows + sills
  • Detailed cobweb removal (corners, ceilings)

What's not included

  • Inside oven or inside fridge (add as add-on or pick move-in/out)
  • Wall washing (add-on per room)
  • Window exteriors / 2nd-story
  • Carpet shampooing / steam cleaning
  • Pet waste, hoarding situations, biohazard

Need any of these? They're available as add-ons in the quote tool — you'll see the price live before you book.
